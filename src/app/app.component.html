Clarity

"Clarity - show cases" is a part of the Clarity Open eGovernment Services project.

The CLARITY project is a two-year project, funded by the European Commission under the Horizon 2020 framework. Grant Agreement number: 693881

"Clarity - show cases" is available as an open source application, maintained by Code for NL on github.com/codefornl/clarity.

Please contact Code for NL for more information.
